linus说过的一句话:

Talk is cheap. Show me the code

就从这个网站：https://github.com/Show-Me-the-Code/show-me-the-code[](https://github.com/Show-Me-the-Code/show-me-the-code) 开始.

第 0001 题：做为 Apple Store App 独立开发者，你要搞限时促销，为你的应用生成激活码（或者优惠券），使用 Python 如何生成 200 个激活码（或者优惠券）？

由于刚看完语法，选择直接抄其代码。由于没有PIL Module，于是云行：

pip install PIL

报错：

Reading http://effbot.org/downloads/#Imaging
No local packages or download links found for pil

找了很久的原因终于在这个网站解决的办法
easy_install -f http://www.pythonware.com/products/pil/ Imaging

然而还是得到一堆warnning，不管它，继续抄代码。修改了很多次，终于成功了。

总结下: python 作为动态语言，相对于之前熟悉的静态语言，很多时候不能提供方法推断，有点让人恐慌。python 拥有非常多的包，这让习惯了严谨的语言人有写不习惯，因为也许很多时候，你需要的仅仅是去找相应的包就可以。

一、CoordinatorLayout的使用
CoordinatorLayout是一个增强型的FrameLayout。它的作用通常就是协调手势效果。例如Toolbar通过固定在顶部，使用CoordinatorLayout就可以在Toolbar滑出屏幕时达到隐藏的效果。另外CoordinatorLayout还要一个可以滑动的子布局，比如RecyclerView、NestedScrollView(ListView、ScrollView不支持。更加说明官方要使用RecyclerView代替ListView. NestedScrollView代替ScrollVIew)。
