每周观察

发表于 2018-11-25   |     |   阅读次数

##自我思考
本周开始准备面试, 打算把View的绘制、Http。RxJava2、Retrofit2等全都了解并明白他们的原理, 作为面试的资料。

这周一直在看HenCoder的View绘制, 之前有看过一次。现在重新看一篇有了更深的理解。这让人想起了一本书《Google工作法》中的。先大略的查看所讲内容, 然后开始思考与自己的所见对比。把知识点纳入自己的思考体系中，建立自己的知识体系。之后在开始细读他们,把他们与自己的经验结合。丰富自己的知识体系。目前我的进度还是很慢的。

##活动

这周在一个碰巧的机会下，去参加了Girls Coding Day活动。教女生Python爬虫编程入门。

有意义的地方

这个是一个有意思的活动, 在教学的过程中了解他们的背景、学历、知识结构、当前的苦恼、还有认知这个世界的角度。与不同背景、行业的人交流其实是一个有意思的事情。也能感觉到他们的焦虑。

####做的不够好的事情
在这次活动的最后一个环节, 由教练和学员上台演讲。回想我在台上的样子，讲的是一点都不好，没有很好的台风，也没有作为一个教练的觉悟。没有自我介绍，没有介绍我们组的情况。其实讲的很失败。上台之前也没有和学员沟通好流程与分工，导致很是错乱。上台之前在下面观察了下, 却没有和学员交流好。

####总结
回想我这些时间的过往, 感觉我都把时间浪费掉了。没有做任何事情。有些后悔。珍惜时间点滴。
