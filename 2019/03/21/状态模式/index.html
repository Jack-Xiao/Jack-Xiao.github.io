状态模式

发表于 2019-03-21   |     |   阅读次数

定义:
允许对象在内部状态发生改变时改变它的行为，对象看起来好像修改了它的类。
场景:
一个类的状态决定了他的行为，而且在运行时必须根据当前的状态来改变行为。

状态模式说策略模式的区别:

策略模式封装的是不同的算法，算法之间没有交互，以达到算法可以自由切换的目的；而状态模式封装的是不同的状态，以达到状态切换行为随之发生改变的目的。这两种模式虽然都有变换的行为，但是两者的目标却是不同的。我们举例来说明两者的不同点。
