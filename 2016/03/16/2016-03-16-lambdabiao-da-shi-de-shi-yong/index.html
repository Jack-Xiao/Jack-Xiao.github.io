lambda表达式的使用

发表于 2016-03-16   |     |   阅读次数

今天有同事问起lambda的用法，以前总是有点似懂非懂。重新看过资料后，发现明白了些。记录下心得，也分享给初学感觉困难的人。

###什么是lambda表达式

以下是来自百度百科的资料：

“Lambda 表达式”(lambda expression)是一个匿名函数，Lambda表达式基
于数学中的λ演算得名，直接对应于其中的lambda抽象(lambda abstraction)，
是一个匿名函数，即没有函数名的函数。Lambda表达式可以表示闭包（注意和数学
传统意义上的不同）。

###一、在android studio中集成lambada
1.在project gradle中增加：

classpath 'me.tatarka:gradle-retrolambda:3.2.4'

2.在module gradle中增加：

apply plugin: 'me.tatarka.retrolambda'

3.在module gradle android中增加

compileOptions{
    targetCompatibility 1.8
    sourceCompatibility 1.8
}

###二、lambda表达式的使用
1.在匿名类中的使用，最典型的就是实现Runnable接口：

new Thread(new Runnable() {
        @Override
        public void run() {
            System.out.println("Hello world");
        }
    });

    new Thread(() -> System.out.println("Hello world"));

2.我们在看一个有参数的

String []at1={"sdafasdf","sdafasdfa","sdfasdf"};
    Arrays.sort(at1, new Comparator<String>() {
        @Override
        public int compare(String lhs, String rhs) {
            return lhs.compareTo(rhs);
        }

        @Override
        public boolean equals(Object o) {
            return super.equals(o);
        }
    });

    Arrays.sort(at1,(String s1,String s2) -> (s1.compareTo(s2)));

不知道你们有没有注意到，似乎这里的接口都只有一个方法。于是我试着写有二个接口的方法测试一下。

    fun5((s) -> System.out.println("s"));
    fun5((s) -> System.out.print("s"));

}

interface TestLambda{
    public void test(String s);

    public int test1(String s,String e);
}

void fun5(TestLambda l){

}

错误提示：

Multiple non-overriding abstract methods....

这似乎容易给人误导，直接看Comparator接口. 然而它有二个抽象方法。

3）知道上面的就可以简单的使用lambda。lambda最强的应该是Java 8中 steam包对于集合的各种操作，可以很方便地并行操作大集合，充分发挥多核CPU潜能。在需要的时候在进一步使用。
