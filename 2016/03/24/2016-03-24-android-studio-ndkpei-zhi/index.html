android studio ndk配置

发表于 2016-03-24   |     |   阅读次数

前言:写久了jdk，尝试着练习一下ndk的编程。

一、搭建NDK环镜

  1. 新建一个空的android project，对着project点击右建，选择open Module Settings。SDK Location
    选项下有Android NDK location:在点击下载(科学上网)
    在gradle.properties最后一行加入 android.useDeprecatedNdk=true

  1. 新建一个类实现native方法

     public class Load {
   public static native String getStringFormC();
}

    然后在Rebuild Project，
    在app/build/intermediates/classes/debug 打开Terminal指令，输入

    javah -jni com.tianyejia.ndkproject.Load

    这时打开classes/debug下面的文件多了一个文件com_tianyejia_ndkproject_ndk_Load.h

  2. 在src/main文件夹下新建文件夹jni,把生成的 .h头文件复制到jni文件夹下。新建一个c类。添加如下代码：

    #include <jni.h>
/*
* Class:     Java_com_wobiancao_ndkjnidemo_ndk_JniUtils
* Method:    getStringFormC
* Signature: ()Ljava/lang/String;
*/

JNIEXPORT jstring Java_com_tianyejia_ndkproject_Load_getStringFormC
(JNIEnv *env, jobject obj) {
return (*env)->NewStringUTF(env, "这里是来自c的string");
}

    注意JNIEXPROT 后面改成自己的包名及类名

  3. 在app中的build.gradle -> defaultConfig括号内添加如下代码
    ndk{
  moduleName "lb" //生成的so名字
  abiFilters "armeabi","armeabi-v7a","x86" //输出指定三种abi体系结构下的so库，目前可有可无。
}

在Load java类中添加
static{
System.loadLibrary(“lb”);
}
5.修改MainActivity 中的代码如下

public class MainActivity extends AppCompatActivity {
  TextView tv;
  @Override
  protected void onCreate(Bundle savedInstanceState) {
  super.onCreate(savedInstanceState);
  setContentView(R.layout.activity_main);
  tv = (TextView) findViewById(R.id.tv);

  tv.setText(Load.getStringFormC());
}

运行结果如图
6.打开app/intermediates/ndk/debug 发现生成了三个夹，对应之前的lb。然后新建libs文件夹把这三个文件夹放进去。
然后删除jni文件试试，运行ok.
