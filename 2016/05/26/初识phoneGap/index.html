初识phoneGap

发表于 2016-05-26   |     |   阅读次数

前言：最近在使用phoneGap, 记录下最近的经验.

PhoneGap 是什么

Phonegap是一款开源的开发框架，旨在让开发者使用HTML、Javascript、CSS等Web APIs开发跨平台的移动应用程序。原本由Nitobi公司开发，现在由Adobe拥有。

官网：http://phonegap.com/

然而，当你按照官方的Guides下载好PhoneGap Developer 和 App并新建一个项目时，你会发现项目中多了很多插件。原来cordova 会按照项目目录下的config.xml 去npm中下载对应的插件。官方送你的大礼包！

常用Cordova 命令

标题为什么不是“常用PhoneGap命令?” 原来phoneGap 几经易主，最后的名称已经被改为Cordova，其内部的插件也都通常称为cordoba-plugin。So 现在所说的Cordova其实就是指代了PhoneGap。

  1. 安装Cordova

    npm install -g cordova

  2. 创建应用程序

    cordova create Hello com.example.hello

其中 Hello 为当前文件夹下创建的项目目录, com.example.hello 为config.xml的中widget的id。用来标示当前应用的唯一性。

  1. 添加平台

    cordova platform add android
    cordova platform and iOS

  2. 插件命令

    cordova plugin add *

    注意，当要加的插件不在npm 中时，可以使用 http路径

    cordova plugin add https://github.com/**********

    这样就可以愉快的使用github上的插件了。

  3. 编译项目

    cordova build android
    cordova build iOS

当提示 pressmied 之类的错误时，是可实行文件的权限问题 执行以下命令

sudo chmod +x <文件>

或者修改读写权限

sudo 777 <文件>

温馨提示，当你第一次build android项目时，需要下载很多的东西。最好找一个网速好的时候，不然就呵呵。。

彩蛋：当我们import 刚build的项目时，总是一闪而过。什么都没有发生。 在试几次还是一样。这都什么鬼？难道新版phonegap 都不能用gradle构建了吗。我们试下别的思路，用import 导入项目试试。然而还是报错，不过终于有反应了。
